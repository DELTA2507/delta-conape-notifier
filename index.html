Funcionamiento

Resumen:

El programa realiza un chequeo constante del calendario de la sección 'Préstamo aprobado' en la página de CONAPE

Si detecta una fecha de cita disponible, se envía una alerta a la página y se reproduce un sonido de alerta.

Al presionar el botón 'Inicializar Notificador', se abrirá una nueva pestaña en la página de CONAPE y el programa se encargara de presionar los botones necesarios de forma automática y continuará con este proceso hasta encontrar una fecha

Una vez que se encuentre una fecha disponible, se le notificará por medio de un sonido de campana proveniente de esta página web (CONAPE Date Notifier) además de una alerta.

Para que seguidamente agende su cita en la fecha disponible
